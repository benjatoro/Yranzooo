Server

Es un sistema operativo diseñado para dar servicios a otras computadoras a través de una red. No está pensado para que alguien lo use sentado frente a él, sino para que esté prendido 24/7 respondiendo pedidos.

  • Optimizado para estabilidad, seguridad y manejar muchas conexiones simultáneas.
  • Muchas veces ni siquiera tiene interfaz gráfica para ahorrar recursos.
  • Atiende a múltiples usuarios o máquinas al mismo tiempo.
  • Hardware enfocado en RAM, almacenamiento y red.
  • Servicios típicos: alojar páginas web, bases de datos, correo, archivos compartidos.

Ejemplos

  • Windows Server (2019, 2022).
  • Ubuntu Server.
  • Red Hat Enterprise Linux.

Workstation

Es un sistema operativo diseñado para un usuario individual que lo usa directamente, sentado frente a la pantalla.

  • Tiene interfaz gráfica pensada para trabajar.
  • Optimizado para tareas del usuario: texto, navegación, diseño, programación.
  • Hardware más enfocado a gráficos y procesador rápido.
  • Un solo usuario activo a la vez.

Ejemplos

  • Windows 11.
  • MacOS en una MacBook.
  • Ubuntu Desktop.

Windows

Sistema operativo desarrollado por Microsoft, lanzado por primera vez en 1985. Es el sistema operativo más usado del mundo en computadoras de escritorio y portátiles.

Características

  • Interfaz gráfica con ventanas, íconos y escritorio.
  • Muy utilizado por su compatibilidad con videojuegos, programas y dispositivos.
  • Software propietario.
  • Gran compatibilidad con hardware de distintas marcas.

Tipos

  • Windows WS (Workstation): pensado para uso personal directo.
  • Windows Server: diseñado para administrar recursos en red.

Comandos comunes

  • dir → ver archivos de la carpeta actual
  • cd [NOMBRE] → cambiar de carpeta
  • cd .. → volver a la carpeta anterior
  • mkdir [NOMBRE] → crear una carpeta nueva
  • tasklist → ver procesos abiertos
  • shutdown → apagar o reiniciar la PC
  • ipconfig → ver configuración de red
  • sfc /scannow → reparar archivos del sistema

macOS

Sistema operativo desarrollado por Apple, exclusivo para sus computadoras. Lanzado en 2001 como Mac OS X, está basado en Unix.

Características

  • Interfaz gráfica pulida con Dock y barra de menú superior.
  • Software propietario: solo se usa en hardware Apple.
  • Muy usado en diseño gráfico, edición de video y desarrollo.
  • Integración con iPhone, iPad y otros dispositivos Apple.
  • Terminal potente basada en Unix.

Tipos

Apple lanza una versión nueva por año con nombre de un lugar de California (Ventura, Sonoma, Sequoia, etc.)

Comandos comunes

  • ls → ver archivos y carpetas
  • cd [NOMBRE] → abrir una carpeta
  • cd .. → volver a la carpeta anterior
  • mkdir [NOMBRE] → crear una carpeta nueva
  • touch [nombre] → crear un archivo vacío
  • mv [origen] [destino] → mover un archivo
  • open . → abrir la carpeta actual en Finder

Linux

Sistema operativo de código abierto y gratuito, creado por Linus Torvalds en 1991. No es un único sistema sino una familia de versiones llamadas distribuciones (distros): Ubuntu, Debian, Fedora, Arch, Mint, etc.

Características

  • Software libre: cualquiera puede ver, modificar y distribuir el código.
  • Gratis en la mayoría de sus distribuciones.
  • Muy usado en servidores y supercomputadoras.
  • Altamente personalizable.
  • Más liviano que Windows.

Tipos

  • Ubuntu: popular y amigable para principiantes.
  • Debian: estable y segura.
  • Fedora: enfocada en software libre.
  • Arch Linux: para usuarios avanzados.

Comandos comunes

  • pwd → ruta del directorio actual
  • cd [directorio] → cambiar directorio
  • ls → listar archivos
  • mkdir [nombre] → crear carpeta
  • touch [nombre] → crear archivo vacío
  • cat [archivo] → mostrar contenido de archivo
  • cp [origen] [destino] → copiar archivos
  • mv [origen] [destino] → mover archivos
  • rm [archivo] → eliminar archivo

Ubuntu

Distribución de Linux desarrollada por Canonical desde 2004. La más conocida para usuarios comunes por ser fácil de instalar y usar.

Características

  • Versión nueva cada seis meses (abril y octubre).
  • Cada dos años se lanza una versión LTS con soporte de 5 años.
  • Sistema Snap: instala programas con todas sus dependencias en una caja cerrada.

Comandos comunes

  • ls → ver carpetas disponibles
  • cd [NOMBRE] → abrir carpeta
  • cd .. → regresar a la carpeta anterior
  • mkdir [NOMBRE] → crear una carpeta nueva
  • touch [nombre] → crear un archivo vacío
  • sudo apt install [programa] → instalar un programa

Emuladores

¿Qué son?

Los emuladores de sistemas operativos son programas que simulan el funcionamiento de un sistema operativo dentro de otro sistema operativo.

¿Cómo funcionan?

Un emulador imita el hardware y/o software de un sistema diferente, traduciendo las instrucciones del sistema emulado para que la máquina real las entienda.

¿Para qué se usan?

  • Correr software de otro SO.
  • Probar sistemas operativos sin instalarlos.
  • Desarrollo de software en distintos sistemas.
  • Retrocompatibilidad con programas viejos.
  • Videojuegos retro.

Herramientas principales

  • VirtualBox: gratis, multiplataforma, el más usado en educación.
  • VMware: más profesional, hay versión paga.
  • Wine: corre programas de Windows en Linux/Mac.
  • WSL: corre Linux dentro de Windows.
  • BlueStacks: emula Android en Windows/Mac.
  • DOSBox: emula MS-DOS para programas viejos.

Emulación vs. Virtualización

  • Emulación: simula hardware diferente (más lento, más flexible).
  • Virtualización: usa el mismo hardware en un espacio aislado (más rápido).
